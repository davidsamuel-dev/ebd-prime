01
Introdução

Contexto e problema

A EBD é um pilar das igrejas evangélicas, mas a gestão semanal ainda enfrenta gargalos operacionais que um sistema dedicado pode resolver.

  • Controle manual de turmas, presença e ofertas em papel ou planilhas
  • Histórico fragmentado — difícil rastrear evasão ou desempenho do aluno
  • Relatórios demorados para consolidar após cada domingo
  • Necessidade de solução mobile, simples e específica para a EBD
02

O que é o EBD Prime?

Ecossistema digital que centraliza turmas, cadastros, chamada, materiais pedagógicos, visitantes, oferta e relatórios — na palma da mão.

Implementação · Backend

Esqueci a senha (RF002)

O app envia usuário e e-mail; a API valida a conta, gera um token único e dispara o link de redefinição via SMTP Hostinger (PHPMailer).

  • Token armazenado como hash SHA-256 — válido por 30 minutos
  • E-mail HTML com link web para nova senha
  • SMTP: smtp.hostinger.com · credenciais no .env
  • Após redefinir, o usuário volta ao app com acesso restaurado

backend/api/auth/forgot-send.php

// RF002 — gera token e envia e-mail SMTP
$plainToken = bin2hex(random_bytes(32));
$tokenHash = hash('sha256', $plainToken);
$expiresAt = (new DateTimeImmutable('+30 minutes'))
  ->format('Y-m-d H:i:s');

$pdo->prepare(
  'INSERT INTO recuperacao_senhas
   (usuario_id, token_hash, expires_at) VALUES (...)'
)->execute([...]);

$resetLink = $baseLink . '?token='
  . rawurlencode($plainToken);

ebd_send_password_reset_email(
  $dbEmail, $resetLink, $congregacaoId
); // PHPMailer → SMTP Hostinger

Deploy e ambiente real

Hostinger

Publicação automatizada do repositório local até produção — pacote PHP + landing enviado por FTP/FTPS para a Hostinger.

01 📦 Empacotar npm run deploy:hostinger
→
02 📤 Upload FTP basic-ftp · FTPS · porta 21
→
03 🖥️ Hostinger hPanel · MySQL · SSL
→
04 🌐 Online ebd.adparaiso.com.br

Script de deploy

prepare-hostinger-deploy.php monta a pasta deploy/hostinger-ebd.adparaiso.com.br/ com API, vendor, landing e .env de produção.

FTP na Hostinger

Credenciais em backend/.env.deploy — host, usuário e senha do FTP do hPanel. Envio recursivo para a raiz do subdomínio.

Validação pós-deploy

Site: ebd.adparaiso.com.br
API: /api/health.php
SMTP: smtp.hostinger.com

  • Um comando: npm run deploy:hostinger — empacota e envia por FTP sem passo manual
  • Infraestrutura Hostinger: hospedagem compartilhada, banco MySQL e certificado HTTPS
  • Landing em / e API em /api/ no mesmo servidor — rotas isoladas (RNF007)
